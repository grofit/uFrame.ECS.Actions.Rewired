Rewired Actions for uFrame ECS
======================================

Targeted Versions
=================

- uFrame ECS beta2
- Rewired 1.0.0.67

Installation
=============
- Make sure you have added this package with at least the Rewired folder
- Make sure you have Rewired installed
- Make sure you have uFrame ECS installed

Using Actions & Conditions
==========================
You should now notice that you have a new category in your actions and conditions called Rewired. 
You can then put the actions into your project and use them rather than interacting progmatically.
